# FastSlow Evo — Sealed Version

## What is finished

FastSlow Evo is now packaged as a usable skill with:
- a clear dual-loop model
- fast/slow routing
- quick-adapt and slow-promote workflows
- starter guides
- templates
- scripts
- example specs
- penetration strategy notes

## Core claim

FastSlow Evo helps an agent improve quickly where local adaptation is enough, and improve carefully where durable promotion is required.

## What it is not

- not a full autonomous self-modification engine
- not a generic office productivity wrapper
- not a vague reflection system with no artifacts

## Recommended first use

Open:
- `references/one-page-start.md`
- `references/quick-adapt.md`
- `references/fast-slow-router.md`

Then run one real incident through the fast loop.
